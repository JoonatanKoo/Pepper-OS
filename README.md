# Pepper-OS     
<img width="537" height="142" alt="image" src="https://github.com/user-attachments/assets/69ee9e99-0563-488d-b075-58343b760733" />

Pepper Operating System is a simple shell-based operating system written in ASM and C.<br>
Pepper is still in early development and i plan on releasing the first release in the summer of 2026.

Current state: PS/2 input, VGA output, Simple shell commands, Programs written in C, App launching from shell, No filesystem, No HDMI support, No DisplayPort support, No USB support,
